响应式 layui 导航扩展组件，支持三级导航，遵循 layui 模块规范。

> Demo: [https://dlj.biz/flynav](https://dlj.biz/flynav)

### 文件说明：
```
flyNav.js     //组件 JS 库
flyNav.js     //组件 css 库
```
### 使用组件目录规范：
```
  ├─layui           //layui 基础框架
  │─layui_exts      //存放第三方组件的目录
      └─flyNav      //flyNav 组件
```
